# 基于Springboot2.0的个人博客网站-myblog

**myblog**是基于[ZHENFENG13](https://github.com/ZHENFENG13)开源项目[My-Blog](https://github.com/ZHENFENG13/My-Blog)，作为自己的个人博客网站，
并且在原项目的基础上正和上自己的东西...


## 更新记录：
* **2018-07-18:** 将原来的项目（[My-Blog](https://github.com/ZHENFENG13/My-Blog)）改造成多模块的项目，并且将springboot版本升级到2.0，解决其中兼容性问题
* **2018-07-30:** 修整项目，并且在其中添加根据“分类”查询相关的文章，解决其中的某一部分bug
* **2018-08-13:** 修复了登录的一个小bug，导致了在登录的时候，后端拦截器拦截了静态资源（js，css，img）
* **持续更新中...**
